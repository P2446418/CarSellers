Just some general instructions for next weeks lab session

to import the project from github into visual studio

- open visual studio and sign in using school email
- File -> Open -> Open from source control
- some window on the right should appear
- choose 'clone'
- enter the url of the gitpage https://github.com/P2446418/CarSellers
- it will ask you to sign in to github
- once project appears open the 'Car_Sellers.sln' file

to connect to the database

- View -> SQL Server Object Explorer
- Right click 'SQL Server' tab -> Add SQL Server
- some window pops up
- Server Name : 'car-sellers-server.database.windows.net'
- change 'Windows Authentication' to 'SQL server authentication'
- Username : 'car_sellers'
- password : 'CSAccess!'
